HTML Tags

<p>

The <p> tag defines a paragraph. 

<p>This is some text in a paragraph.</p>

This is some text in a paragraph.

<div>

The <div> defines a division in an HTML document. It is used as a container for HTML elements. 


        <div>
         <h2>This is a heading in a div element</h2>
         <p>This is some text in a div element.</p>
        </div>

This is a heading in a div element

This is some text in a div element.

<button>

The <button> tag defines a clickable button. You can put text inside a <button> element. 

<button>SUBSCRIBE</button>

<a>

The <a> tag defines a hyperlink, which is used to link from one page to another. 

<a href="https://www.google.com/" target="_blank">Link to Google</a>
Link to Google

<h1> - <h6>

The <h1> to <h6> tags are used to define HTML heading. The h1 defines the most important heading. h6 defines the least important heading. 


        <h1>This is a heading 1</h1>
        <h2>This is a heading 2</h2>
        <h3>This is a heading 3</h3>
        <h4>This is a heading 4</h4>
        <h5>This is a heading 5</h5>
        <h6>This is a heading 6</h6>

<ul>

The <ul> tag defines an unordered (bulleted) list. The <li> tag is used to define each list item. 


        
    <h4>List of desserts:</h4>
  
      <ul>
        <li>Brownies</li>
        <li>Cake</li>
        <li>Pie</li>
        <li>Pudding</li>
        <li>Ice-cream</li>
      </ul>

List of desserts:

  • Brownies
  • Cake
  • Pie
  • Pudding
  • Ice-cream

<ol>

The <ol> tag defines an ordered list. An ordered list can be numerical or alphabetical. 


      <h4>Steps on fixing a flat tire:</h4>
    
        <ol>
          <li>Remove the valve cap from the tire's valve stem.</li>
          <li>Attach the fixed flat canister to the valve stem.</li>
          <li>Release the sealant into the tire by pressing the button or trigger on the canister.</li>
          <li>Remove the canister and drive the vehicle briefly to distribute the glue inside the tire.</li>
          <li>Check the tire's pressure and adjust as needed.</li>
        </ol>

Steps on fixing a flat tire:

  1. Remove the valve cap from the tire's valve stem.
  2. Attach the fixed flat canister to the valve stem.
  3. Release the sealant into the tire by pressing the button or trigger on the canister.
  4. Remove the canister and drive the vehicle briefly to distribute the glue inside the tire.
  5. Check the tire's pressure and adjust as needed.

<img>

The <img> tag is used to embed an image in an HTML page. 


        <img src="redvelvetcake.jpg" alt="Red Velvet Cake"></img>

CSS Properties

CSS properies define the style and layout of HTML elements.

color

The color property sets the color of text.

color: blue;

font-size

The font-size property sets the size of the font.

font-size: 12px;

font-family

The font-family property specifies the font for an element.

font-family: 'Times New Roman', Times, serif;

background-color

The background-color property sets the background color of an element. 

background-color: lightblue;

border-width

The border-width property sets the width of an elements's four borders. 

border-width: 5px;

border-style

The border-style property sets the style of an element's four borders. 

border-style: solid;

border-color

The border-color property sets the color of an element's four borders. 

border-color: pink;

border-radius

The border-radius property defines the radius of the element's corners. 

border-radius: 25px;

padding

An element's padding is the space between its content and its border. The padding property sets the padding for all four sides of an element. 

padding: 20px;

margin

The margin property sets the margin for all four sides of an element.

margin: 30px;

height

The height property sets the height of an element.

height: 25px;

width

The width property sets the width of an element.

width: 15px;

box-shadow

The box-shadow property attaches one or more shadows to an element.

box-shadow: 5px 5px 5px rgba(0, 0, 0, 0.15);

opacity

The opacity property sets the opacity level for an element.

opacity: 0.5;
